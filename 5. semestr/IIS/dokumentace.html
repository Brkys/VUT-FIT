Implementace

Implementace probíhala v kombinaci bootstrapu s frameworkem laravelem (v 5.4)
Databáze je napsána v MySQL.
Soubor /resources/views/layouts/app.blade.php obsahuje základní rozvržení stránky, které je pak importováno do ostatních stránek
Složka /resources/views obsahuje php soubory s obsahem pro jednotlivé stránky (nastavení účtu, administrace, práce s kriminálnímí činnostmi, stránku s územími, ...)
Složka /app/Http/Controllers obsahuje ovladače (kontrolery)
Složka /app obsahuje modely
Složka /config obsahuje soubory s nastavením aplikace (debug mód, databáze, ..)
Složka /routes obsahuje cesty (speciální odkazování na stránky)

Instalace

  • Je třeba nainstalovat laravel
  • PHP v7.1.3
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension
  • BCMath PHP Extension
  • Pro vytvoření databáze stačí zprovoznit mysql databázi, nastavit v config/database.php ip hostujícího serveru, port, jméno databáze, jméno uživatele a heslo

Známé problémy

Vzhledem k první zkušenosti s laravelem implementace nemusí být dokonalá. Prosíme o trpělivost při přistupování k databázi; hosting je dost pomalý. Usoudili jsme, že je zbytečné uchovávat, zda li je uživatel Caporegime, nebo Řádový člen, toto jsme specifikovali pouze v tabulce users ve sloupci permission, kde číslo se uchovává číslo práv; čím vyšší práva, tím vyšší pozice (až na výjimku s právem 5, což je admin) a uchovávat, zda-li je činnost vraždou, což stačí napsat v popisku kriminální činnosti. Bohužel jsme až poslední den zjistili, že laravel má debugování v sobě, tak jsme byli omezeni nevypsáním chybových hlášek a museli hledat vše ručně, což byl důvod, proč jsme neimplementovali aliance a pozvání na setkání donů.
